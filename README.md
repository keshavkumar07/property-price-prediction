# Property Price Prediction

A professional-looking property price prediction web app built with Flask and a RandomForest model.

## Setup

1. Create virtual env:

```bash
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate # mac/linux
2.	Install dependencies:
pip install -r requirements.txt
3.	Put your dataset at data/House Price Prediction Dataset.csv.
4.	Train model:
python train_model.py
5.	Run app:
python app.py
Open http://127.0.0.1:5000
